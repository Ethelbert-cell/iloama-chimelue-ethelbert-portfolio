## 🏗️ Running Locally

```bash
# Clone the repository
git clone https://github.com/Ethelbert-cell/iloama-chimelue-ethelbert-portfolio.git
cd iloama-chimelue-ethelbert-portfolio

# Install dependencies
npm install

# Start the dev server
npm run dev
```

> **Note:** You'll need Node.js installed. Copy `.env.example` to `.env.local` and fill in your `GEMINI_API_KEY` if required.
